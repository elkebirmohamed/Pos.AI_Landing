# POS.AI Landing Page

Landing page commerciale pour la solution de caisse enregistreuse intelligente POS.AI.

## Technologies

- **Framework** : React 18
- **Build Tool** : Vite
- **Langage** : TypeScript
- **Styles** : Tailwind CSS (via CDN pour la démo, configurable via PostCSS pour la prod)
- **Icônes** : Lucide React

## Démarrage Local

1. Installer les dépendances :
   ```bash
   npm install
   ```

2. Lancer le serveur de développement :
   ```bash
   npm run dev
   ```

## Déploiement

Ce projet est prêt pour un déploiement sur [Vercel](https://vercel.com).

1. Poussez ce code sur un dépôt GitHub.
2. Importez le projet dans Vercel.
3. Vercel détectera automatiquement Vite et déploiera l'application.

Pour plus de détails, consultez le [Guide de Déploiement](./Docs/Docs.md).
